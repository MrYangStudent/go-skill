# Prompt Master - 提示词大师

基于吴恩达《AI Prompting for Everyone》课程精华的对话式提示词构建技能。

## 核心功能

- **对话式提示词构建** — 通过六步结构化对话逐步引导用户生成高质量提示词
- **提示词诊断优化** — 对已有提示词进行完整性检查和优化建议
- **场景适配** — 支持写作、编程、数据分析、学习教育等场景的特定引导
- **进阶技巧注入** — 自动融入角色扮演、思维链（CoT）、少样本学习、格式控制等

## 六步工作流

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 🎭 角色定义 | 确定 AI 扮演的角色 |
| 2 | 📖 背景描述 | 提供任务上下文 |
| 3 | 🎯 目的阐述 | 明确期望目标 |
| 4 | 🔒 约束设定 | 定义限制条件 |
| 5 | 📋 输出格式 | 指定呈现方式 |
| 6 | 💡 示例引导 | 提供参考案例（可选） |

## 触发方式

直接输入以下关键词即可触发技能：

```
帮我写一个提示词
优化这个prompt
提示词工程
写prompt
构建提示词
prompt template
向AI提问
```

## 使用示例

### 从零构建

```
用户：帮我写一个提示词，用于生成产品功能介绍。
AI：好的！让我通过六步对话一步步帮你构建...

第一步：你希望 AI 扮演什么角色...
...
```

### 优化已有

```
用户：帮我优化这个提示词："写一篇关于AI的文章"
AI：让我诊断一下这个提示词...

| 维度 | 状态 |
|------|------|
| 角色定义 | ❌ 未指定 |
| 目的阐述 | ❌ 太模糊 |
| ...
```
